# membran

layer I — a React Native macOS app.

## Quick start (macOS)

```bash
git clone <repo-url> && cd membran
bash scripts/setup-macos.sh
```

Then in two terminals:
```bash
npm run start:macos          # Terminal 1 — Metro bundler
npx react-native run-macos   # Terminal 2 — build and launch
```

See [CLAUDE.md](./CLAUDE.md) for full setup details, known gotchas, and conventions.
